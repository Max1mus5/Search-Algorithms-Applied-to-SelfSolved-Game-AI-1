# Requirements for 8-Puzzle Search Algorithms (FASE 1)
# This project uses only standard Python libraries

# Core dependencies (included in Python standard library):
# - typing (Python 3.5+)
# - time (builtin)
# - sys (builtin)
# - os (builtin)
# - importlib (Python 3.4+)

# Optional dependencies for GUI (main_UI.py):
# - tkinter (usually included with Python, may need separate installation on some Linux systems)
#   Install with: sudo apt-get install python3-tk (Ubuntu/Debian)
#               : sudo yum install tkinter (CentOS/RHEL)
#               : sudo pacman -S tk (Arch Linux)

# No external packages required via pip!
# This implementation follows the requirement of using only standard Python libraries.

# To run the project:
# 1. Terminal interface: python main.py
# 2. Quick test: python main.py --test  
# 3. GUI interface: python main_UI.py (requires tkinter)

# Tested with Python 3.8+
